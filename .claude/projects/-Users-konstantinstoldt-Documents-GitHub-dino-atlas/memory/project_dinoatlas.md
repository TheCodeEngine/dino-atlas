Dino-Atlas ist eine Lern-App fuer Konstantins Kinder Karl (4) und Oskar (6). Die Kids lieben Dinos. Eine fruehere Pokemon-Go-aehnliche Version wurde nicht gespielt weil draussen echtes Pokemon Go gewonnen hat.

**Why:** Die Kinder sollen spielerisch ueber Dinos lernen, aber nicht zu lange am Bildschirm haengen. Offline-Aktivitaeten (Malen, Basteln) sind ausdruecklich gewuenscht.

**How to apply:** Immer auf begrenzte Bildschirmzeit achten, altersgerechte Schwierigkeitsgrade (Karl=einfach, Oskar=anspruchsvoller), TTS fuer alles weil Karl noch nicht lesen kann. Content-Generierung via Google Gemini (Key vorhanden). Plattform: iPhone.
