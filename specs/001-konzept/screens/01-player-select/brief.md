# Spieler-Auswahl

## Zweck
"Wer ist heute dabei?" - Kinder auswaehlen die mitspielen. Jedes Kind bekommt seine eigene Expedition.

## Kontext
- Erster Screen den die Kinder sehen
- Wird jeden Tag aufgerufen
- Eltern oder aeltere Kinder bedienen ihn

## Elemente
- [ ] Ueberschrift: "Wer geht heute auf Expedition?" (vorgelesen)
- [ ] Forscher-Charakter mit Sprechblase
- [ ] Forscher-Avatare der Kinder (gross, antippbar):
  - Jedes Kind hat einen eigenen Avatar (Dino-Forscher-Stil)
  - Name unter dem Avatar
  - Ausgewaehlt = leuchtender Rand / Haekchen
  - Nicht ausgewaehlt = leicht ausgegraut
- [ ] "Los geht's!" Button (erst aktiv wenn mind. 1 Kind ausgewaehlt)
- [ ] Anzeige: Welches Kind hat heute schon gespielt (kleines Icon)
- [ ] Zugang zum Eltern-Dashboard (dezentes Icon oben rechts, z.B. Zahnrad)

## Verhalten
- Antippen auf Avatar = auswaehlen/abwaehlen (Toggle)
- Mehrere Kinder gleichzeitig auswaehlen moeglich
- "Los geht's!" startet Expeditionen fuer alle ausgewaehlten Kinder
- Wenn ein Kind heute schon fertig ist: Avatar zeigt "muede" State

## Stimmung
- Aufregung und Vorfreude! "Wer kommt mit?"
- Warme Farben, grosse Avatare, einladend
- Forscher ist begeistert und wartet auf die Kinder
