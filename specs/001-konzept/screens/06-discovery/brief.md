# Entdeckung!

## Zweck
Der grosse Jubel-Moment! Der Dino wird enthuellt mit echtem Bild, Comic-Version und Steckbrief.

## Kontext
- Hoehepunkt der Expedition
- Dino landet im Museum
- Soll sich wie Weihnachten anfuehlen

## Elemente
- [ ] Grosses Dino-Bild (erst realistisch, dann auch Comic-Version umschaltbar)
- [ ] Dino-Name gross und prominent:
  - Deutsch: "Dreihorn-Gesicht"
  - Lateinisch: "Triceratops"
- [ ] Forscher jubelt: "Unglaublich! Ein Triceratops! Den suche ich schon seit Jahren!"
- [ ] Steckbrief-Karte mit Icons:
  - Erdzeitalter (mit kleinem Zeitleisten-Icon)
  - Ernaehrung (Pflanze/Fleisch Icon)
  - Groesse (Vergleich: "So gross wie 2 Autos!")
  - Gewicht
  - Wo gelebt (Mini-Weltkarte oder Kontinent-Icon)
- [ ] Fun-Fact Blase ("Wusstest du? Der Triceratops hatte 800 Zaehne!")
- [ ] TTS-Button: Steckbrief wird vorgelesen
- [ ] Konfetti / Sterne Animation im Hintergrund
- [ ] "Ab ins Museum!" Button
- [ ] Toggle: Echtes Bild ↔ Comic-Version

## Verhalten
- Screen baut sich stueckweise auf (Dramatik):
  1. Erst Silhouette/Schatten
  2. Dann Bild enthuellt (Slide/Fade)
  3. Name erscheint
  4. Steckbrief gleitet rein
- Alles wird automatisch vorgelesen
- "Ab ins Museum!" speichert den Dino und geht zum Offline-Auftrag

## Stimmung
- JUBEL! Feier! Stolz!
- Goldene Akzentfarben, Konfetti, Sterne
- Das Kind soll sich fuehlen wie ein echter Entdecker
